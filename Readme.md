# FreelanceHub

A modern, full-featured freelance marketplace platform built with **Next.js 14**, **Supabase**, and **Tailwind CSS**.

## ✨ Features

- User authentication & role-based access (Client / Freelancer)
- Browse & search freelancers
- Post projects and receive applications
- **Detailed Freelancer Profiles** (`/freelancers/[id]`)
- **Project Details Pages** (`/projects/[id]`) with application management
- Personalized user dashboard
- Reviews & ratings system
- Responsive, beautiful UI

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database & Auth**: Supabase
- **Icons**: Lucide React
